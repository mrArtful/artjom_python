# Recursive Lesson: Building Rock Paper Scissors

Follow these steps to understand how to build a persistent CLI application.

### Steps:
1.  **`01_interaction.py`**: Getting started with `questionary` for inputs.
2.  **`02_logic.py`**: Adding game rules and randomness.
3.  **`03_persistence.py`**: Learning how to read/write JSON files.
4.  **`04_loop_and_save.py`**: Combining the game loop with file saving.
5.  **`05_final_app.py`**: Adding stats and cleaning up the code.

### How to use:
Run each file in order to see the application grow!
```bash
python steps/01_interaction.py
...
python steps/05_final_app.py
```
